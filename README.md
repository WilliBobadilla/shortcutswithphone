# Atajos de teclado :keyboard:

Estos atajos de teclados son basicamente pensados en las personas zurdas, que usan el mouse con la zurda, debido a que copiar, pegar, guardar son un poco tediosas y tambien el control que queda a la derecha esta un poco encimada y queda lejos algunas teclas con las que se combina. Entonces la idea es tener unos atajos de teclado en la pantalla de tu telefono(proximamente hare la version fisica con Arduino )
